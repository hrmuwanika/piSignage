# piSignage Release Notes

Release notes have been split into three sub-documents.

## Server

- **[RELEASE NOTES - Server.md](<RELEASE NOTES - Server.md>)** — covers both server lines:
  - **3.x.x** — legacy server
  - **5.x.x** — modern rewrite (server2)
  
  Most recent releases ship to both lines simultaneously and use a dual version notation (e.g. *"3.9.6/5.0.5 Server Release"*).

## Player

- **[RELEASE NOTES - Player 5.x.x.md](<RELEASE NOTES - Player 5.x.x.md>)** — Player2 (4.x.x onwards), the rewritten player line. Use this for all current Bookworm/Trixie based images.
- **[RELEASE NOTES - Player 3.x.x.md](<RELEASE NOTES - Player 3.x.x.md>)** — the original (legacy) player line, covering 3.x.x and earlier (down to 0.7.5).
